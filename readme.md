My Cozy Place

Overview
A cozy 3D web scene built for a monthly creative challenge. Features custom-designed assets, warm lighting, and interactive elements using Three.js.

Live Demo
https://my-cozy-place.vercel.app/

Features
Creative concept and references inspired by cozy interiors

Custom 3D modeling and scene composition

Interactive rendering with Three.js

Optimized asset loading for smooth web performance

Technologies Used
Three.js

Blender (for modeling assets)

JavaScript/HTML/CSS

Vercel (for deployment)
